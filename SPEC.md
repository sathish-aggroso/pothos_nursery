# Pothos Nursery - Website Specification

## Project Overview
- **Project Name**: Pothos Nursery
- **Type**: Multi-page static brochure website (Tailwind CSS)
- **Core Functionality**: Showcase nursery services, plants, pricing, and contact information
- **Target Users**: Home gardeners, landscaping enthusiasts, local community

## UI/UX Specification

### Visual Design (Pastel Earthy)

**Color Palette**
- Primary: `#7C9A82` (sage green - pastel)
- Secondary: `#B8A394` (warm taupe - pastel)
- Accent: `#D4C4A8` (sand - pastel)
- Background: `#F5F2EB` (warm cream - pastel)
- Text: `#3D3D3D` (soft black)
- Light: `#E8E4DC` (light beige)
- Card BG: `#FDFCFA` (off-white)
- Muted: `#A8B5A0` (muted sage)

**Typography**
- Headings: "Cormorant Garamond" (serif, elegant)
- Body: "DM Sans" (clean, readable)
